# Portfolio-Power-BI-Project
Этот проект представляет собой Power BI-отчёт для анализа ключевых бизнес-метрик компании Plant Co., включая продажи, количество и прибыль от продаж, с основным акцентом на сравнении Year-to-Date и Prior-Year-to-Date. Дашборд служит комплексным инструментом для оценки динамики по странам, продуктовым категориям, аккаунтам и временным периодам, позволяя руководству и командам быстро отслеживать ключевые показатели, выявлять риски, понимать причины отклонений и находить точки роста.
